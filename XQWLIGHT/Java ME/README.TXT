=== 安装提示 ===
1. 不要对 XQWLight.jar 解压缩，而是直接复制到手机上安装！
2. 对于大多数支持 Java ME 的手机，安装 XQWLight.jar 即可，部分手机可能还需要 XQWLight.jad。
3. 如果您的手机支持触摸屏而没有按键，那么请选用 XQWLight-NoKeyPad.jar 和 XQWLight-NoKeyPad.jad。
