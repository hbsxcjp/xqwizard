=== 安装提示 ===
1. 手机象棋游戏套件包含两款产品，分别位于目录“象棋小巫师”和“象棋巫师魔法学校”下。
2. 如果您的手机支持触摸屏而没有按键，那么请选用带“无按键”的目录。
3. 不要对 .jar 文件解压缩，而是直接复制到手机上安装。
4. 对于大多数支持 Java ME 的手机，安装 .jar 即可，部分手机可能还需要 .jad。
