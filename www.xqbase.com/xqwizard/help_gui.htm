如何切换界面的大小？
　　象棋巫师(包括魔法学校)有两种界面，小界面适合800x600分辨率的显示模式，大界面适合1024x768或更高分辨率的显示模式，设置方法如下：
　
　　选项=>设置=>界面=>大/小；
　
　　设置完以后，退出并重新启动象棋巫师或魔法学校，设置才会生效。
如何修改音效？
　　象棋巫师(包括魔法学校)在摆放和搬动棋子时会产生丰富的音效，这些声音来自SOUNDS目录下的若干WAV文件，用户可以根据需要替换它们。
　　各种事件产生的声音跟WAV文件的对应关系如下：
　
　　　　点击　　 => CLICK.WAV
　　　　非法着法 => ILLEGAL.WAV
　　　　走子　　 => MOVE.WAV
　　　　吃子　　 => CAPTURE.WAV
　　　　升变　　 => PROMOTE.WAV
　　　　将军　　 => CHECK.WAV
　　　　电脑走子 => MOVE2.WAV
　　　　电脑吃子 => CAPTURE2.WAV
　　　　电脑升变 => PROMOTE2.WAV
　　　　电脑将军 => CHECK2.WAV
　　　　赢棋　　 => WIN.WAV
　　　　和棋　　 => DRAW.WAV
　　　　输棋　　 => LOSS.WAV
　　　　新的棋局 => NEWGAME.WAV
如何关闭音效和背景音乐？
　　关闭音效的设置方法如下：
　
　　选项=>音效。
　
　　关闭背景音乐的设置方法如下：
　
　　选项=>音乐=>无声。
